# Simple Banking System in C++

An object-oriented Console Application designed for basic banking operations. This project showcases key concepts of Object-Oriented Programming (OOP) in modern C++, utilizing smart pointers (`std::shared_ptr`) for clean memory management.

## 🚀 Features Implemented
* **Customer Management:** Create individual customer profiles linked with multiple account spaces.
* **Account Management:** Open discrete bank accounts featuring unique identifiers and tracked balances.
* **Core Ledger Functions:** Complete support for **Deposits**, **Withdrawals**, and direct **Fund Transfers** between accounts.
* **Transaction History:** Real-time generation and reporting of detailed transaction logs per account.

## 🛠️ Technical Elements Illustrated
* Class Composition and Data Encapsulation
* Standard Library Container use (`std::vector`, `std::string`)
* Precision text formatting using `<iomanip>` (with strict namespace scoping `std::fixed`)
* Smart Pointers (`std::make_shared`) to eliminate resource leaks
